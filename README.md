# telco-customer-churn-analysis
📊 Telco Customer Churn Analysis | Python | SQL | Data Visualization  Analyzed 7,043 customer records to identify churn patterns. Found that month-to-month contracts have 42%+ churn rate vs 3% for 2-year contracts. Built interactive dashboard with key business recommendations.  Tools: pandas, numpy, matplotlib, seaborn, SQLite, Jupyter
